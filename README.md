# DevCard

Uma aplicação móvel desenvolvida para criar cartões de visita digitais de forma simples e elegante para desenvolvedores. O app permite que você insira seus dados, como nome, cargo, empresa e tecnologias favoritas, e gere um cartão virtual personalizado.

## 🛠️ Principais Tecnologias

* **React Native**: O framework principal para a construção da interface móvel.
* **Styled Components** *(se você usou)*: Para a estilização dos componentes de forma organizada.
* **React Navigation** *(se você usou)*: Para a navegação entre as telas da aplicação.
