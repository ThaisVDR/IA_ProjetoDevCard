# DevCard

Um app mobile desenvolvida para criar cartões de visita digitais de forma simples. O app permite que você insira seus dados, como nome, cargo, empresa e tecnologias favoritas, e gere um cartão virtual personalizado.

## Principais Tecnologias

* **React Native**: O framework principal utilizado para a construção da interface móvel nativa (Android e iOS).
* **Expo**: A plataforma e conjunto de ferramentas que facilita o desenvolvimento, testes e execução do app React Native sem complicações.
